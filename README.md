# 贪吃蛇 (Snake Game)

网页版贪吃蛇小游戏，使用纯 HTML/CSS/JavaScript 实现。

## 操作方法

| 按键 | 方向 |
|------|------|
| W | 上 |
| A | 左 |
| S | 下 |
| D | 右 |
| 空格 | 重新开始 |

## 游戏机制

- 使用 **WASD** 控制蛇的移动方向，不可反向移动
- 每吃到一个食物（红色方块），蛇身增长一节，得分 **+10**
- 随着得分增加，蛇的移动速度会逐渐加快
- 撞墙或撞到自己身体时游戏结束
- 游戏结束后按**空格键**或点击**重新开始**按钮继续
- 最高分会保存在浏览器本地存储中

## 技术栈

- HTML5 Canvas
- CSS3
- 原生 JavaScript

## 本地运行

直接用浏览器打开 `index.html` 即可。
